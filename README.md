# live-itrip  project

admin 管理后台
common 公共类库
email 邮件服务
file 文件服务
redis 缓存服务
sso
web 网站
wechat 微信sdk封装

技术：
1. spring + springmvc + mybatis + maven
2. bootstrap

4. Shiro
5. Dubbo Zookeeper

2. web 网站，已基本实现页面展示机后台内容编辑功能
